# 💎 Future Design: 資産運用 & キャリア設計シミュレーター

「今の生活」から「将来の資産」、そして「今やるべき行動（副業）」までを一本道で繋ぐ、人生設計シミュレーターです。

## 🌟 主な機能
- **📝 家計診断**: 月々の固定費から、無理のない投資可能額を算出。
- **👴 老後シミュレーション**: インフレ率やNISA、取り崩しまで考慮した一生の資産推移を可視化。
- **🎯 キャリア逆算**: 目標金額から必要な月収を逆算し、不足額に応じた具体的な副業を提案。

## 🚀 技術スタック
- **Language**: Python 3.11+
- **Framework**: Streamlit
- **Data Analysis**: Pandas, Numpy
- **Design**: Custom CSS for UI/UX improvement

## 🛠️ インストールと実行方法
1. リポジトリをクローン:
   git clone https://github.com/YOUR_ID/money-simulator.git
   cd money-simulator

2. 仮想環境の作成と起動:
   python -m venv .venv
   source .venv/bin/activate

3. ライブラリのインストール:
   pip install -r requirements.txt

4. アプリの起動:
   streamlit run app.py
